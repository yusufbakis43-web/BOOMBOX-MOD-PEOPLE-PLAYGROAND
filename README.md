# 🔥 BOOMBOX MOD 🔥

BOOMBOX, oyun deneyimini tamamen değiştirmek için hazırlanmış özel bir müzik modudur. İçerisinde internet kültürü, meme sesleri, agresif edit müzikleri ve eğlenceli karışımlar bulunur. Normal boombox sisteminden sıkılan oyuncular için hazırlanmıştır.

## 🎵 MOD İÇERİĞİ

Bu mod paketinin içinde bulunan şarkılar:

1. KURMANCI - HERE GULE
2. I NEED MORE BULLETS
3. ÇOMAK SOKARIM
4. MANIFEST SNAP
5. SAMSUNG OVER THE HORIZON
6. FEYZULLAH
7. CARRY POTER ROBUX

Tüm sesler özel olarak seçilmiş olup edit, troll ve eğlence amaçlı kullanılabilir.
